How to Use This Widget

The Competition Configuration Tool can be embedded in any webpage with just a few lines of code:

Step 1: Add a target element

<div id="competition-config"></div>

Step 2: Include the widget script

<script src="widget.js"></script>

Step 3: Initialize the widget

<script>
  CompetitionConfig.init('competition-config');
</script>

Using via jsDelivr CDN (once published to GitHub)

<script src="https://cdn.jsdelivr.net/gh/username/repo@main/widget.js"></script>
<script>
  CompetitionConfig.init('competition-config');
</script>
